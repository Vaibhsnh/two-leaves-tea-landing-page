# 🍃 Two Leaves and a Bud — Responsive Landing Page Clone

A modern, fully responsive recreation of the **Two Leaves and a Bud** landing page, built using only **HTML5** and **CSS3**. This project focuses on precise layout replication, clean semantic structure, responsive design principles, and modern frontend development practices.

Designed with an editorial-inspired aesthetic, the website adapts seamlessly across desktop, tablet, and mobile devices while maintaining visual consistency and accessibility.
